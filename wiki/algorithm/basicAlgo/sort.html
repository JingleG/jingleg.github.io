Sort

排序也是经典算法，虽然绝大多数时候不需要手写，但不排除出现的可能。
常用排序算法：

  • 插入
  • 快排，及其变形快速选择
  • 归并
  • 三色排序

JDK 中 sort() 实现

  • 长度小于MAX_INSERTION_SORT_SIZE，插入排序
  • 根据 元素结构/数量/是否重复 使用 快排/双轴快排/归并/堆排/计数

Insert Sort

  • Time: $ \mathcal{O}( n^2 ) $
  • space: $ \mathcal{O}( 1 ) $
  • 稳定，数据有序时效率高，best: $ \mathcal{O} (n) $

Quick Sort

  • Time: average: $\mathcal{O}( n\log{n} )$, worst: $ \mathcal{O} ( n^2 ) $

  • space: $ \mathcal{O}( \log{n} ) $

  • 优化

    • 三数取中，避免极端数据

    • 序列较短时使用插入排序

    • 将与分界元素相等的元素聚集在分界元素周围， 处理大量相同元素的序列

      • 三路（双轴）快排 3-way Radix（DualPivot） Quicksort

        结合荷兰国旗问题，随机选取分界点 mid 后，将数列划分为 小于mid，等于mid，大于mid

quick select 快速选择

选择top k元素
平均 $ \mathcal{O} (n) $

优化: BFPRT 最坏 $ \mathcal{O} (n) $

Merge Sort

  • Time: $\mathcal{O}( n\log{n} )$
  • space: $ \mathcal{O}( n ) $
