Git

用户配置

  • 全局配置：在 ~/.gitconfig 中，默认使用的配置文件
  • 项目配置：在项目工作目录中 .git/config，若存在则覆盖全局的配置
  • 系统配置：应用于系统中所有用户的默认配置

使用 --global 则配置全局，否则配置目录下的项目

查看配置

SSH

  • -t 表示ssh的密钥类型，常用 rsa
  • -C 标识名称
  • -b 密钥长度 RSA密钥，最小768，默认2048
  • -f 指定密钥对文件名

公钥 .pub 上传，私钥留在本地
